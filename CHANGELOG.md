# Change Log

All notable changes to the "react-refactor-to-file" extension will be documented in this file.

## [0.0.2] - 2025-12-05

### Added

- Support for selecting custom destination path for extracted components.
- Automatic test file generation with configurable strategies (`relative` or `interactive`).
- `help` command to show available commands and settings.

### Changed

- Refactored codebase to use service-oriented architecture for better maintainability.

## [0.0.1] - 2024-12-05

- Initial release of React Refactor to File.
